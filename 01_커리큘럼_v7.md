# AI CEO School — 10주 커리큘럼 v7

> 이해하고 → 검증하고 → 확산한다
> CEO가 직접 검증한 AI를 조직 전체로 확산하는 전사 AI 전환 커리큘럼

---

## Phase 1 — 전환 전략 수립 (1~3주)

### Week 1. AI 시대, 조직이 바뀌는 법
CEO가 알아야 할 AI의 큰 그림과, 실제로 조직 전체를 바꾼 기업들의 전사 전환 사례를 봅니다.

**학습 내용**
- 생성형 AI → 에이전트 → 바이브코딩 — 3가지 변화의 흐름
- AI Native 기업의 전사 전환 사례 (뤼튼, 업스테이지, 채널톡, 당근)
- CEO의 "구축 / 구매 / 무시" 의사결정 프레임워크

**산출물**
- ChatGPT·Claude로 경영 시나리오 3가지 체험
- AI 기회 캔버스 초안 (템플릿)
- 업종별 테이블 토론 + 네트워킹

---

### Week 2. 전사 확산 — 우리 조직에 맞는 모델 찾기
AI를 조직에 퍼뜨리는 4가지 방법론을 배우고, 리스크를 이해한 뒤, 우리 조직의 확산 전략 초안을 세웁니다.

**학습 내용**
- 전사 확산 4가지 모델
  - AI TF 구성 → 전사 확대
  - 부서별 챔피언(전파자) 지정
  - 전 직원 의무 교육 + 실습
  - 파일럿 부서 → 성과 검증 → 전사 확대
- 데이터 보안·개인정보·저작권·환각 — CEO가 알아야 할 리스크
- AI 거버넌스 — 도입 전 조직이 갖춰야 할 것

**산출물**
- "우리 조직 확산 전략 초안" 작성 (템플릿)
  - 확산 모델 선택 + 이유
  - 파일럿 대상 부서/업무 후보
  - 예상 장애물 + 대응 방안
- 업무 후보별 리스크·실현가능성 평가표

---

### Week 3. 바이브코딩 첫 체험 + 실행 계획
"말로 설명하면 앱이 된다"를 Cowork으로 직접 체험하고, 향후 7주간 조직에 적용할 실행 계획을 세웁니다.

**학습 내용**
- 바이브코딩이란? — 코드를 몰라도 업무 도구를 만드는 시대
- Cowork 소개 — CEO가 바로 쓸 수 있는 AI 데스크탑앱
- 프롬프트 설계 원칙 (역할, 맥락, 형식, 제약)

**산출물**
- Cowork 바이브코딩 실습 (90분)
- 확산 대상 업무 Top 3 최종 확정
- 실행 계획서 (어떤 업무를 → 어떤 도구로 → 언제까지)

---

## Phase 2 — 도구 검증과 조직 적용 (4~7주)

### Week 4. CEO 실전 활용 — 보고서 분석 & 리서치
대표가 매일 하는 일에 AI를 직접 써봅니다. "이거 진짜 되네"를 체감하는 주차.

**학습 내용**
- 보고서 검토에 AI 활용
  - 팀 보고서 던지면 → 핵심 인사이트 추출 + "이거 왜 이러지?" 질문 목록 자동 생성
  - 재무제표·매출 데이터 → 트렌드 분석 + 이상치 탐지
- 시장/경쟁사 자동 리서치
  - "우리 업계 AI 도입 현황" 리서치 리포트 자동 생성
  - 경쟁사 동향 모니터링 자동화
- 미팅 준비 자동화
  - 내일 거래처 미팅 → 상대 기업 분석 + 예상 질문 + 어젠다 초안

**산출물**
- Cowork으로 본인 업무 3가지에 AI 적용 실습 (90분)
- "가장 효과 있었던 활용법" 1개 선정 → 프로토타입 v1

---

### Week 5. Skill 만들기 + Claude Code
4주차에서 써본 것 중 반복되는 업무를 Skill로 만들어, "나만 쓰는 것"에서 "팀이 쓰는 것"으로 전환합니다.

**학습 내용**
- Skill이란? — 반복 업무를 패키징해서 누구나 쓸 수 있게 만드는 단위
- Claude Code + 에이전트 체험 — "지시하면 알아서 실행하는 AI" 데모
- Cowork vs Claude Code — 각각 언제 쓰는가
- 실습: 4주차에서 만든 활용법을 Skill로 변환

**산출물**
- Claude Code 체험 실습 (60분)
- Skill 설계서 — 이름, 입력, 출력, 대상 사용자 정의
- 프로토타입 v2 (Skill 형태로 업그레이드)

---

### Week 6. 조직 확산 설계 + ROI
만든 Skill을 어떤 부서에, 어떤 순서로 퍼뜨릴지 설계하고, 투자 대비 효과를 측정합니다.

**학습 내용**
- Skill 공유 체계 설계 — 누가 만들고, 누가 쓰고, 누가 관리하는가
- 부서별 배포 전략 — 파일럿 → 확대 로드맵
- ROI 산출법 — 도입 전/후 비용 비교, 시간 절감 측정법
- AI 실습 플랫폼 활용 — 팀원용 실습 자동 생성

**산출물**
- Skill Top 5 설계서 + 부서별 배포 계획
- ROI 산출 워크숍 (60분) → ROI 표 완성
- 확산 전략 초안(Week 2) 업데이트 → v2

---

### Week 7. 중간 발표 — 조직 적용 현황
한 달간 조직에서 AI를 어떻게 적용했는지 공유하고, 피드백을 받습니다.

**학습 내용**
- 발표 구조: "계획한 것 → 시도한 것 → 조직 반응 → 배운 것"
- 3가지 카테고리: Pioneer (적극 실행) / Explorer (탐색 중) / Planner (전략 수립 중)

**산출물**
- 그룹 내 발표 → Best-of 쇼케이스
- 전문가 피드백 + Phase 3 개선 목표 설정

---

## Phase 3 — 확산 완성 (8~10주)

### Week 8. 프로토타입 개선 + 게스트 CEO
피드백을 반영하여 Skill을 개선하고, 실제로 전사 AI 도입에 성공한 CEO의 이야기를 듣습니다.

**학습 내용**
- Skill 개선 전략 — "CEO만 쓰는 것 vs 팀 전체가 쓸 수 있는 것"
- 게스트 CEO 특강 — AI 전사 도입 성공 사례 + Q&A
- 네트워킹: 게스트 CEO + 수강생 교류

**산출물**
- Claude Code + AI 실습 플랫폼에서 Skill 개선 (60분 실습)
- 개선된 Skill + 부서별 확산 실험 결과 정리

---

### Week 9. 확산 회고 + 로드맵 재설계
실제 전사 적용 경험에서 출발합니다. 부서별 확산의 성공/실패를 돌아보고 전략을 재설계합니다.

**학습 내용**
- 부서별 확산 결과 회고 — 무엇이 됐고, 무엇이 안 됐는가
- 직원 저항 극복 — 변화 관리 3가지 전략
- AI 전담팀 vs 전사 내재화 — 우리에게 맞는 다음 단계
- Skill 공유 체계 고도화 — 지속 운영 구조

**산출물**
- 전사 확산 로드맵 v3 (3/6/12개월)
- 피어 리뷰 — CEO끼리 서로의 확산 로드맵 리뷰 + 보강

---

### Week 10. 최종 발표 + 수료
10주간의 전사 AI 전환 여정을 정리하고, 실제 결과를 발표합니다.

**학습 내용 (발표 항목)**
- AI 도입 배경과 목표
- CEO가 직접 활용한 사례와 결과
- 조직 확산 현황 — 몇 개 부서, 몇 명이 쓰고 있는가
- ROI 분석 + 향후 12개월 로드맵

**산출물**
- 그룹 내 발표 → Best-of 쇼케이스
- 시상: "최고 전환 여정" / "가장 용감한 피봇" / "최고 조직 확산"
- 수료식 + 네트워킹 디너
